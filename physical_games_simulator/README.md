# Physical Games Simulator

You said you wanted physical games back.

Physical Games Simulator is a parody-but-functional Python utility that recreates the classic PC gaming experience of the 1990s and early 2000s.

The program generates:

- A physical disc package
- A unique disc token
- A launcher that checks for that token before starting a game

No DRM is bypassed.
No game files are modified.
No software is cracked.

This is not copy protection.

It is a simulation.

The goal is simple:

If modern gaming suddenly returned to the era of "Insert Disc to Continue", would people actually enjoy it?

Or do they simply miss the idea of physical ownership?

## What It Does

1. Select a DRM-free game.
2. Generate a physical disc package.
3. Burn the package to a DVD, Blu-ray, USB drive, or other removable media.
4. Launch the game through the generated launcher.
5. If the matching disc token is not present, the launcher refuses to start the game.

In other words:

No disc.

No game.

## Why This Exists

Physical Games Simulator started as a joke.

Then it became a social experiment.

The software explores the tradeoff between:

- Ownership
- Preservation
- Archival practices
- Convenience
- Nostalgia

Many players say they want physical games back.

This tool lets them try it.

## Important

This utility:

- Does NOT remove DRM
- Does NOT bypass DRM
- Does NOT modify games
- Does NOT provide copy protection

It is intended for DRM-free software and preservation-oriented workflows.
